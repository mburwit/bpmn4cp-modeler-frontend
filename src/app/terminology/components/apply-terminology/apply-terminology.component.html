Apply Terminology

Code systems can be used to provide value sets for specific properties of elements in the modeler. Changes of the code systems will be immediately selectable in the modeller, where the code system is used. Changes of the code systems, esp. renaming code system concepts, will not automatically be applied to the content of existing models, where code system concepts are used. But this can be triggered here.

Experimental feature! Use at your own risk! We recommend backing up the models (export BPMNXML) before applying the terminology.
